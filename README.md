# Nova Bank — UX/UI Case Study

A comprehensive, interactive UX/UI case study for **Nova Bank**, a modern mobile banking application. 

This repository contains the custom HTML and CSS used to build the single-page case study presentation. It features:
- A custom dark-mode design system (Indigo/Violet palette).
- Interactive scroll reveals and glassmorphism UI components.
- Integrated usability testing metrics and user flow diagrams.
- Print-optimized CSS for PDF exporting.
